# EP Restore Theme

## Beschreibung

Dies ist eine 1:1 Wiederherstellung des Elektriker Pohl Designs aus dem Website-Backup. Das Theme wurde vollständig in deutschem Text erstellt und behält alle ursprünglichen Design-Elemente, Layouts und Inhalte bei.

## Theme-Struktur

### Hauptdateien
- `style.css` - Haupt-Stylesheet mit WordPress Theme-Header
- `functions.php` - Theme-Funktionen und Setup
- `header.php` - Header-Template mit Logo und Navigation
- `footer.php` - Footer-Template mit Links und Copyright
- `front-page.php` - Homepage-Template
- `index.php` - Standard-Template für Archiv-Seiten
- `page.php` - Template für statische Seiten
- `single.php` - Template für einzelne Beiträge

### Template Parts
- `template-parts/services.php` - Dienstleistungen-Sektion
- `template-parts/about.php` - Über-uns-Sektion
- `template-parts/contact.php` - Kontakt-Sektion mit Formular
- `template-parts/problems.php` - Typische Probleme-Sektion

### Assets
- `assets/css/` - Zusätzliche Stylesheets (falls benötigt)
- `assets/js/main.js` - JavaScript für Interaktivität
- `assets/img/` - Alle Bilder vom Original-Backup

## Installation

1. Laden Sie das Theme in den Ordner `wp-content/themes/` hoch
2. Aktivieren Sie das Theme im WordPress-Dashboard unter "Design > Themes"
3. Das Theme ist sofort einsatzbereit

## Anpassungen

### Logo ändern
Gehen Sie zu "Design > Customizer > Site Identity" und laden Sie Ihr eigenes Logo hoch.

### Telefonnummer ändern
Suchen Sie nach `+4915777406869` in allen Theme-Dateien und ersetzen Sie sie durch Ihre Nummer.

### Farben anpassen
Bearbeiten Sie die CSS-Variablen in `style.css`:
- Hauptfarbe (Schwarz): `#1a1a1a`
- Akzentfarbe (Grün): `#61CE70`
- Textfarbe: `#666`
